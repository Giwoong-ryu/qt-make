# 한국어 자막 분할 가이드 (Korean Subtitle Segmentation Guide)

> 최종 업데이트: 2026-01-17
> 참조: Netflix Timed Text Style Guide, KSS 3.0, WhisperX v3

---

## 1. 개요

한국어 자막은 영어와 달리 **종결어미/연결어미** 구분이 핵심입니다.
잘못된 위치에서 자막을 끊으면 의미 전달이 어색해집니다.

### 핵심 원칙

| 우선순위 | 규칙 | 설명 |
|----------|------|------|
| 1 | 문장부호 | `.!?` → 무조건 끊음 |
| 2 | 연결어미 | `-서`, `-니까`, `-는데` → 절대 끊지 않음 |
| 3 | 종결어미 | `-습니다`, `-해요`, `-했어` → 끊어도 됨 |
| 4 | 접속부사 | `그리고`, `하지만` → 앞에서 끊음 |

---

## 2. 타이밍 기준 (Netflix 가이드라인)

### CPS (Characters Per Second)

| 대상 | CPS | 설명 |
|------|-----|------|
| 성인 | 16-20 | 일반 콘텐츠 |
| 아동 | 13 | 어린이 콘텐츠 |
| 설교/교육 | 14-16 | 집중 청취 콘텐츠 |

### 자막 길이

| 항목 | 값 | 비고 |
|------|-----|------|
| 최소 지속시간 | 0.83초 (5/6초) | 너무 짧으면 읽기 어려움 |
| 최대 지속시간 | 7초 | 너무 길면 집중력 저하 |
| 최대 글자수 | 7초 × 16CPS = 112자 | 한 자막당 |
| 화면당 줄 수 | 최대 2줄 | 가독성 유지 |

## 3. 종결어미 (Sentence Endings)

> 이 패턴으로 끝나면 **문장이 완결됨** → 끊어도 됨

### 격식체 (가장 확실)

```
습니다, 입니다, 됩니다, 합니다, 갑니다, 옵니다
십시오, 십니다, 셨습니다, 었습니다, 았습니다
겠습니다, 이었습니다, 셨어요, 였습니다
```

### 존댓말

```
세요, 시죠, 시네요, 으세요, 으시죠
으십시오, 하세요, 주세요, 보세요, 가세요, 오세요
```

### 해요체

```
해요, 네요, 죠, 어요, 아요, 예요, 에요
겠죠, 잖아요, 거예요, 건가요, 나요, 을까요
했어요, 됐어요, 였어요, 이에요, 데요, 래요
볼게요, 할게요, 줄게요, 갈게요, 올게요
봐요, 해봐요, 가요, 와요, 줘요
```

### 반말 (확실한 것만)

```
했다, 됐다, 었다, 았다, 인다, 한다, 이다
간다, 온다, 본다, 던다, 준다, 쓴다
했어, 됐어, 었어, 았어, 거야, 이야, 해
가, 와, 봐, 줘  # 짧은 반말 종결
```

### 의문형

```
습니까, 입니까, 는가, 은가, 냐, 나, 까
을까, 할까, 볼까, 갈까
할래, 볼래, 갈래, 할까요, 볼까요
```

### 청유형/명령형

```
읍시다, 합시다, 하자, 해라, 가자, 보자
하세요, 가세요, 오세요, 보세요
```

### 감탄형

```
구나, 군요, 네, 구만, 더라, 던데요
```

## 4. 연결어미 (Connecting Endings)

> 이 패턴으로 끝나면 **문장이 이어짐** → 절대 끊지 않음
>
> **주의**: 1글자 패턴(`고`, `는`, `은`, `면`)은 오탐이 많아 제외

### 원인/이유

```
해서, 에서, 어서, 아서, 라서, 이라서
니까, 으니까, 니깐, 으니깐
때문에, 탓에, 덕분에
```

### 대조/양보

```
지만, 는데, 은데, 더니, 던데
는데도, 은데도, 지만은
```

### 동시/나열

```
면서, 으면서, 으며, 고서, 고는
```

### 목적/의도

```
려고, 으려고, 으러, 려면, 으려면
```

### 조건/가정

```
으면, 거든, 다면, 라면, 이라면
```

### 정도/비교

```
도록, 게끔, 듯이, 처럼, 만큼, 대로
```

### 인용 (중요!)

```
다고, 라고, 냐고, 자고
다는, 라는, 다니, 라니
```

### 보조적 연결

```
아서, 어서, 여서, 아도, 어도, 여도
하게, 하지  # "~하게 되다", "~하지 않다"
```

## 5. 접속부사 (Conjunctions)

> 이 단어가 나오면 **앞에서 끊음** (새 문장 시작 신호)

### 순접 (이어지는 흐름)

```
그리고, 그래서, 그러므로, 따라서, 그러니까, 그러니
```

### 역접 (반대 흐름)

```
그러나, 하지만, 그렇지만, 그런데, 다만, 반면에
```

### 전환

```
그래도, 어쨌든, 아무튼, 한편, 그나저나
```

### 예시/부연

```
예를 들어, 즉, 다시 말해, 특히, 물론
```

## 6. 특수 도메인: 성경/설교

### 고유명사 분리 방지

다음 단어들은 중간에서 끊지 않도록 주의:

```
# 지명
예루살렘, 베들레헴, 갈릴리, 나사렛, 가나안, 이스라엘

# 인명
아브라함, 이삭, 야곱, 모세, 다윗, 솔로몬
바울, 베드로, 요한, 마태, 누가, 마가

# 종교 용어
바리새인, 사두개인, 십자가, 부활절, 성령, 예배
```

### 성경 인용 패턴

```
# 장절 표기 - 끊지 않음
"요한복음 3장 16절"
"마태복음 5:3-12"
"창세기 1:1"
```

### 설교 특유의 종결

```
# 추가 종결어미
아멘, 할렐루야
~하옵소서, ~하옵나이다  # 기도문
```

## 7. 구현 예시 (Python)

```python
class KoreanSubtitleSegmenter:
    """한국어 자막 분할기"""

    # 설정
    TARGET_CPS = 16
    MAX_DURATION = 7.0  # 초
    MIN_DURATION = 0.83  # 5/6초

    SENTENCE_ENDINGS = (
        # 격식체
        '습니다', '입니다', '됩니다', '합니다',
        # 해요체
        '해요', '네요', '어요', '아요', '예요',
        # 반말
        '했다', '됐다', '거야', '이야',
        # 의문형
        '습니까', '을까요', '할까',
        # ... (전체 목록은 위 참조)
    )

    CONNECTING_ENDINGS = (
        # 2글자 이상만!
        '니까', '는데', '지만', '면서',
        '다고', '라고', '려고', '으면',
        # ... (전체 목록은 위 참조)
    )

    CONJUNCTIONS = (
        '그리고', '그래서', '하지만', '그런데',
        '따라서', '그러나', '그래도',
    )

    def is_sentence_end(self, text: str) -> bool:
        """문장 종결 여부 판단"""
        text = text.strip()
        if not text:
            return False

        # 1. 문장부호 → 무조건 끊음
        if text[-1] in '.!?':
            return True

        # 2. 연결어미 → 절대 끊지 않음
        for ending in sorted(self.CONNECTING_ENDINGS, key=len, reverse=True):
            if text.endswith(ending):
                return False

        # 3. 종결어미 → 끊어도 됨
        for ending in sorted(self.SENTENCE_ENDINGS, key=len, reverse=True):
            if text.endswith(ending):
                return True

        return False

    def should_break_before(self, word: str) -> bool:
        """이 단어 앞에서 끊어야 하는지"""
        return word in self.CONJUNCTIONS
```

## 8. 검증 체크리스트

자막 분할 결과를 검증할 때:

- [ ] 연결어미(`-니까`, `-는데`, `-서`)로 끝나는 자막이 없는가?
- [ ] 1글자 패턴(`고`, `는`)으로 끝나는 자막이 없는가?
- [ ] 접속부사 앞에서 적절히 끊어졌는가?
- [ ] CPS가 16-20 범위 내인가?
- [ ] 자막 지속시간이 0.83초 ~ 7초 범위인가?
- [ ] 고유명사가 중간에서 끊어지지 않았는가?
